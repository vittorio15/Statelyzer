## Getting Started
Fork the repository

Clone the repository:
```bash
git clone https://github.com/vittorio15/Statelyzer
```

Install the dependencies:
```bash
cd Statelyzer
```
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
